**Nidhi-AI-Agent**

A conversational AI system that acts as my professional digital twin, enabling recruiters to query my skills, experience, projects, and technical depth in real time.

Key features:
LLM-based contextual response generation
Structured resume-to-knowledge transformation
Intelligent prompt engineering
Scalable backend architecture
Secure API deployment

Designed to demonstrate applied AI engineering, system design, and practical LLM integration.
